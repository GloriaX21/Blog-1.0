# NatureBlog 1.0 - Django Blog Application

This is a Blog Web Application, written in <strong>Django 3</strong>. <strong>This application allows the creation and publication of posts</strong>. The posts are listed in the main layout, and it's possible to see the Title, the Published Date, and a small part of its Text Content. By clicking on the Post's Link, it's possible to see all the content of this post.
